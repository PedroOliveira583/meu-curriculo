# Currículo em HTML com Iframes

Este é o currículo de **Pedro Augusto Menezes de Oliveira**, feito para a disciplina de Desenvolvimento Web.

## 🧠 Descrição

Cada parte do currículo foi separada em arquivos HTML e carregada dinamicamente com `<iframe>` usando navegação por links.

## 📁 Estrutura

- `curriculo.html` — Página principal com iframe
- `objetivo.html` — Objetivo profissional
- `identificacao.html` — Dados pessoais
- `formacao.html` — Formação acadêmica
- `softskills.html` — Habilidades interpessoais
- `hardskills.html` — Habilidades técnicas
- `experiencia.html` — Experiências e projetos

## 🚀 Como acessar

1. Suba os arquivos em um repositório público no GitHub.
2. Ative o GitHub Pages.
3. O link será:  
   `https://seuusuario.github.io/meu-curriculo/`

> Substitua “seuusuario” pelo seu nome de usuário no GitHub.
